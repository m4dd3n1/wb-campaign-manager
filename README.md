# 🎯 Wildberries Campaign Manager

Веб-приложение для анализа рекламных кампаний Wildberries и автоматического удаления кластеров с высоким CPO (Cost Per Order).

## ✨ Возможности

- 📊 **Анализ кампаний** - анализирует CPO каждого кластера
- 🗑️ **Автоматическое удаление** - удаляет кластеры через API Wildberries
- 💾 **История анализов** - сохраняет все результаты в БД
- ⚙️ **Настраиваемые пороги** - выбирайте свой лимит CPO
- 🎨 **Веб-интерфейс** - удобный UI для управления кампаниями

## 📋 Требования

- Python 3.8+
- pip
- Интернет соединение
- API ключ Wildberries

## 🚀 Установка

1. **Клонируй репозиторий**
```bash
git clone https://github.com/m4dd3n1/wb-campaign-manager.git
cd wb-campaign-manager
```

2. **Создай виртуальное окружение**
```bash
python -m venv venv

# На Windows
venv\Scripts\activate

# На macOS/Linux
source venv/bin/activate
```

3. **Установи зависимости**
```bash
pip install -r requirements.txt
```

4. **Создай файл .env** (опционально)
```bash
cp .env.example .env
```

## 🔑 Получение API ключа Wildberries

1. Перейди на [https://seller.wildberries.ru/](https://seller.wildberries.ru/)
2. Авторизуйся в личном кабинете
3. Перейди в **Настройки → API и интеграции → API токены**
4. Создай новый токен с правами на управление кампаниями
5. Скопируй токен (он понадобится при использовании приложения)

Пример API ключа: `eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9...`

## 💻 Запуск приложения

```bash
python main.py
```

Приложение запустится на `http://localhost:8000`

Открой браузер и перейди по адресу выше.

## 📖 Как использовать

1. **Введи API ключ** - вставь свой токен Wildberries
2. **Укажи ID кампаний** - введи ID кампаний через запятую
3. **Установи порог CPO** - укажи максимальный CPO в процентах (по умолчанию 10%)
4. **Нажми "Запустить анализ"** - приложение автоматически:
   - Получит данные кампаний
   - Проанализирует кластеры
   - Удалит кластеры с CPO выше порога
   - Сохранит результаты

## 📊 API Endpoints

### POST /api/analyze
Запускает анализ кампаний

```json
{
  "api_key": "ваш_api_ключ",
  "campaign_ids": ["123456", "789012"],
  "cpo_threshold": 10.0
}
```

### GET /api/analysis-history
Возвращает историю анализов

### GET /api/analysis/{analysis_id}/clusters
Возвращает удаленные кластеры для анализа

### GET /api/campaigns
Возвращает список кампаний

## 🗄️ База данных

Приложение использует SQLite с тремя таблицами:

- **analysis_runs** - запись о проведенных анализах
- **deleted_clusters** - информация об удаленных кластерах
- **campaigns** - информация о кампаниях

БД хранится в файле `wb_campaigns.db`

## ⚠️ Важно

- **API ключ хранится безопасно** в вашем браузере, не отправляется на третьи сервисы
- **Будьте осторожны** при установке низкого порога CPO - это может привести к удалению многих кластеров
- **Протестируйте** на нескольких кампаниях перед массовым использованием
- **Рекомендуемый порог CPO** - 10% от цены товара

## 🐛 Troubleshooting

### Ошибка: "Не может подключиться к API Wildberries"
- Проверь, верно ли указан API ключ
- Убедись, что у ключа есть права на управление кампаниями
- Проверь интернет соединение

### Ошибка: "Порт 8000 занят"
```bash
# Используй другой порт
python main.py --port 8001
```
